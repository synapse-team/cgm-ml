# Dataset-Database

## How to use CLI

### Create an initial, empty database

```bash
python dsdb_cli.py init
```


### Update existing database with ETL data

```bash
python dsdb_cli.py update
```

### Filter PCDs.

```bash
python dsdb_cli.py filterpcds
```

### Sort PCDs.

```bash
python dsdb_cli.py sortpcds
```
